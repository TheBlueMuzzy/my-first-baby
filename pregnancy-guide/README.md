# Pregnancy & First Year Guide

A phase-by-phase reference covering pregnancy (first visit through birth) and the first year. Built as discrete markdown files for use in a CLI app and for a Claude Code agent to read.

## Scope & assumptions
- US-based care, standard ACOG/CDC schedule. Location context: NYC.
- Pregnancy dated the standard way: ~40 weeks counted from the first day of the last menstrual period (LMP), not from conception.
- Reference only. An OB, midwife, or pediatrician's instructions override anything here.

## Time-sensitive content
Vaccine schedules and food advisories change. Items in this guide were verified on 2026-06-29 against ACOG, CDC, and AAP sources. Re-verify before relying on any vaccine timing or food rule. See `CLAUDE.md` for the current vaccine-schedule caveat.
