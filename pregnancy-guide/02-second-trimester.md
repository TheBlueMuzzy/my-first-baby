# Second Trimester (Weeks 14-27)

## Medical checkpoints
- Routine visits every ~4 weeks: weight, blood pressure, fundal height, fetal heart rate, urine.
- Quad/multiple-marker screen (~wk 15-20): blood test, used if NIPT was not done.
- Amniocentesis (~wk 15-20): diagnostic, only if screening flags risk.
- Anatomy scan / level 2 ultrasound (~wk 18-22): detailed organ check, placenta position, sex usually confirmable. The big one.
- Glucose challenge test (~wk 24-28): screens for gestational diabetes.
- Flu and COVID vaccines: recommended during pregnancy, any trimester.

## What's happening (mom)
- "Honeymoon" stretch: nausea usually eases, energy returns.
- First movement ("quickening") felt ~wk 18-22.
- Bump becomes visible; possible round ligament pain, congestion, heartburn.

## Do
- Continue prenatal vitamin; provider may add iron or vitamin D.
- Moderate exercise fine (walking, swimming, prenatal strength/yoga).
- Begin sleeping on side (left preferred) as belly grows.
- Dental checkup (safe now; gum health affects pregnancy).
- Start maternity-leave conversation with HR.

## Avoid
- All first-trimester food/substance rules still apply (alcohol, high-mercury fish, unpasteurized items).
- Lying flat on back for long stretches later in this window.
- Overheating, contact sports, fall-risk and high-impact activities.

## Partner (dad) tasks
- Attend the anatomy scan (~wk 20).
- Co-decide on learning the sex or not.
- Start the registry together; research car seat, stroller, crib.
- Tour and shortlist pediatricians + daycares now (NYC waitlists run long).
- Book a childbirth class for late second / early third trimester.
- If gestational diabetes is positive, support the diet/monitoring changes.

> Reference only. Provider overrides. Re-verify vaccine timing before relying on it.
