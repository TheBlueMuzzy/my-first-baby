# Master Partner (Dad) Checklist

## First Trimester (wk 1-13)
- [ ] Take over cat litter, heavy lifting, harsh-chemical chores
- [ ] Attend first prenatal visit + dating scan
- [ ] Stock nausea-friendly food (crackers, ginger, bland snacks)
- [ ] Set up shared calendar + running appointment-question list
- [ ] Respect her announcement timeline; don't leak

## Second Trimester (wk 14-27)
- [ ] Attend the anatomy scan (~wk 20)
- [ ] Co-decide on learning the sex
- [ ] Start registry; research car seat, stroller, crib
- [ ] Tour + shortlist pediatricians and daycares (NYC waitlists run long)
- [ ] Book a childbirth class
- [ ] Support diet/monitoring if gestational diabetes is positive

## Third Trimester (wk 28-40+)
- [ ] Map hospital route + backup, parking, after-hours entrance
- [ ] Install car seat (~wk 35) and get it inspected
- [ ] Pack your own hospital bag
- [ ] Learn the contraction go-threshold cold (e.g., 5-1-1)
- [ ] Know the birth plan; be ready to advocate
- [ ] Set the call tree, pet/childcare, work coverage

## Labor
- [ ] Time contractions; trigger the go decision
- [ ] Drive or call the ride; bags + car seat at the door
- [ ] Communicate with staff; advocate for her preferences
- [ ] Hold the call tree until she's ready

## Fourth Trimester (0-12 wk)
- [ ] Take night shifts, diapering, bottle feeds, burping
- [ ] Own household + meals; be the visitor gatekeeper
- [ ] Watch mom for physical warning signs + postpartum depression
- [ ] Take paternity leave; do skin-to-skin
- [ ] Support the feeding choice without judgment
- [ ] Admin: birth certificate, SSN, add baby to insurance (tight deadlines)

## Months 3-12
- [ ] Split night wakings fairly; agree on sleep approach
- [ ] Lead childproofing before baby is mobile (~6 mo)
- [ ] Do solo caregiving stretches (be competent, not a helper)
- [ ] Track both parents' mental load; rebalance openly
- [ ] Manage records + next admin round (daycare, etc.)

## Standing rules (all phases)
- [ ] Provider/pediatrician instructions override any general guidance, including this
- [ ] When in doubt on a warning sign, call. Newborn fever under 3 months is always an ER trip.
